Phase 1

Create your Supabase account and project

Supabase is a free, open-source database service. You sign up once; everything else happens inside the project you'll create in the next step.

  1. Go to supabase.com and click "Start your project".
  2. Sign up with GitHub (recommended — one less password to remember) or use an email and password.
  3. Verify your email if prompted. You'll be taken to the Supabase dashboard.

A Supabase project is your private database in the cloud. It takes about 2 minutes to spin up after creation.

  1. From the dashboard, click "New project".
  2. Give it a name — suggest rec.
  3. Set a strong database password. Use the generator below, then save it in your password manager. Never commit it to the repo.
  4. Choose a region: select eu-west-2 (London) if available, otherwise eu-central-1 (Frankfurt).
  5. Leave the plan as Free. Click "Create new project" and wait ~2 minutes.
